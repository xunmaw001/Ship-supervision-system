# Ship-supervision-system
船舶监造系统是基于Spring Boot与Vue框架的专业化数字平台。后端Spring Boot确保系统稳健运行，数据处理高效；前端Vue打造直观易用的操作界面。该系统覆盖船舶设计审查、建造进度追踪、质量控制、文档管理等核心功能，支持实时数据同步与多维度分析。旨在提升船舶监造效率，确保建造质量，优化资源配置，为船东及船厂提供透明、高效的船舶监造服务。
